## Cleanup
You have just finished work on a new `feature`, `bug-fix`, or `refactor`. In oder to cleanup the workspace, perform the following:
- Ensure all the planning documents are maked as 'complete' or 'done'
- Move the planning documents over into `planning/archive`
- If not already done, commit all the code to the working branch
- Push the code, Merge and Squash. Note: leave code from other agents alone - just operate on this branch/worktree
- Once all the new changes are safely on Main, execute a final cleanup step by removing the empty working branch and/or worktree

# App Backend Structure and Design

The backend is written in Python (3.11) and uses the FastAPI framework to serve requests and interact with the Databases.

## 1. Overview:

- Python 3.11: Primary language and version
  - Dependencies: `backend/requirements.txt`
- FastAPI: API route handler framework
- SQLAlchemy: Database ORM
- Alembic: Database migrations
